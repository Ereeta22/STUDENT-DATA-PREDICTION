# STUDENT-DATA-PREDICTION

📌 Project Overview

This project presents an interactive Student Performance Analysis Dashboard built using Microsoft Power BI. The dashboard analyzes various factors that influence student exam scores such as attendance, study hours, sleep patterns, family income, and physical activity.

The goal of this project is to transform raw student data into meaningful insights that help understand academic performance trends and support better decision-making.

🎯 Objectives
Analyze student behavior and academic performance
Identify key factors affecting exam scores
Visualize patterns in attendance, sleep, and study habits
Provide data-driven insights for improving student outcomes
🛠️ Tools & Technologies
Power BI
Data Cleaning & Transformation (Power Query)
Data Modeling
DAX (Data Analysis Expressions)

📂 Dataset Description
The dataset includes the following features:

Gender
Attendance
Hours Studied
Sleep Hours
Tutoring Sessions
Physical Activity
Family Income
School Type
Distance from Home
Previous Scores
Motivation Level
Internet Access
Exam Score
📊 Dashboard Features
🔹 Key Metrics (KPIs)
Average Exam Score
Total Attendance
Total Tutoring Sessions
Average Sleep Hours
Hours Studied

🔹 Visualizations
📌 Distance vs School Time (Bar Chart)
📌 Gender Distribution (Donut Chart)
📌 Family Income vs School Type (Area Chart)
📌 Physical Activity vs Exam Score (Bar Chart)
📌 Sleep Hours vs Exam Score (Line Chart)

📈 Key Insights
Students with balanced sleep (6–8 hours) tend to score higher
Increased study hours improves performance but only up to a limit
Physical activity positively impacts exam scores
Students living closer to school show better attendance
Family income influences access to better school types

🚀 Business Problem
Educational institutions often struggle to understand:

Why some students perform better than others
How lifestyle and environment affect learning
What factors should be prioritized to improve results

This dashboard helps solve these problems using data-driven insights.

💡 Solution
By using Power BI:

Data is cleaned and transformed into a structured format
Relationships are built between variables
Interactive visuals provide easy analysis
Insights help educators make informed decisions
